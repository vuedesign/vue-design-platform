### 入门

快速入门

#### 环境要求

- node 16.x: [官网](https://nodejs.org/en/)
- pnpm 7.x: [官网](https://pnpm.io/)


#### 安装

克隆项目
```bash

# 创建工作空间且进入
mkdir workspace && cd workspace

# 克隆项目
git clone https://github.com/vuedesign/vue-design-platform

# 进入项目根目录
cd vue-design-platform

# 安装项目依赖
pnpm install

# 启动项目
pnpm run dev
```
